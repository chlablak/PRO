User guide

Intent

This guide is intended for Graphy users. It contains a quick start guide on how to declare a graph, add edges and vertices and apply algorithms to it. You will also be able to save and load your work and export it to image formats. This document is not designed to explain the intricate details of how each of the algorithms work, since this information can easily be found online.
