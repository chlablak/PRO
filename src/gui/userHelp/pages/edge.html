Edge

Declaration

Very much like a vertex, an edge can be simply declared as a connection between two vertices:

e = (0--1);

but it can also include information such as weight, a label, max capacity and min capacity. These properties must be separated by a : colon and are all optional. Here are two examples of how this can be done, using only the label property and using the max capacity and min capacity:

e = (0--1:"myFirstEdge");

e = (0--1:::5:2);

the complete grammar for declaring an edge is:

variableName = (firstVertex--SecondVertex:"label":weight:maxCap:minCap);


Edge ID

Each edge has an ID that is local to the connection. A connection is a symbolic relationship between two vertices. This means there can be as many edges with the same ID as there are pairs in a graph. This ID is mainly used to differentiate edges when removing one from a connection containing many edges.


Adding edges to a graph

To add an edge to a graph, the two vertices connected to this edge must already be present. This rule is applicable for anonymous declaration between the { } braces and also for addition to the graph using the += operator. This means the following declaration:

g = {0,1,1--2,2,3};

is not valid, and should instead be written:

g = {0,1,2,1--2,3}; or g = {0,1,2,3,1--2};

The += operator works the same way as with vertices. You can therefore add anonymous edges:

g += (0--1);

as well as edges previously declared:

g += e;


Modifying an existing edge

Modifying the properties of an edge can be done using the += operator. To do so, you must specify the vertices that define the edge and its local ID with the [ ] brackets, which should be visible in the graphical representation. For example:

g = {#3};

g += (0--1::5);

g += (0--1[0]::3);

will change the weight of the current edge from 5 to 3.

If the ID specified does not exit (0 in the example above), this command will have the effect of adding a new edge to the graph.

Note: parameters are passed by value, so modifying them after adding them to the graph will not have any influence. More on this here.


Removing edges from a graph

Removing all the edges between two vertices is accomplished with the -= operator, e.g.:

g -= (0--1);

If you with however to remove one of many parallel edges, you can do so using their ID, for example:

g -= (0--1[1]);

And you can also remove many edges from different connections, using the [ ] brackets:

g -= [(0--1[0]), (1--2)];


Directed edges

As well as -- regular edges shown above, a graph can be composed of -> or <- directed edges. Adding one of these to a graph already containing regular edges will change the graph type to a directed graph and create two directed edges of opposite direction between the appropriate vertices. For example:

dg = {#3, 0--1};

dg += (1->2);

will create the same graph as:

dg = {#3, 0->1, 1->0, 1->2};

All the rules specified above also apply to these directed graphs.
