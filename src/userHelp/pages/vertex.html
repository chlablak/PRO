Vertex

Declaration

A vertex can be declared with anything from a simple ID, to a label, weight, max capacity and min capacity. The most basic vertex is simply declared with:

v = (1);

If you wish to create vertices with more information you can do so, by separating the information with a : colon, according to the following grammar:

variableName = (ID:"label":weight:maxCap:minCap);

for example:

v = (0:"myFirstVertex":3:5:2);

If you only need to add max capacity and min capacity, you can avoid unused information, by lining up : colons.

v = (0:::5:2);

But if you just need a label, there is no need to add the subsequent : colons:

v = (0:"myFirstVertex");


Adding vertices to an existing graph

Once your graph is created you can add a single vertex with:

g += v;

or add many vertices with:

g += [v,w,x];

it is also possible to add anonymous vertices to a graph like so:

g += (1);

or add many, even mixing the notation:

g += [(1), (2:"myOtherVertex"), w];


Modifying an existing vertex

The properties of a vertex can be modified using the += operator. For this, you should specify the ID of an existing vertex and then its properties, as if adding a new vertex, e.g.:

g = { };

g += [(0::5),(1::5)];

g += (1::3);

will have the effect of changing the weight of the vertex with ID 1 from 5 to 3.

Note: parameters are passed by value, so modifying them after adding them to the graph will not have any influence. More on this here.


Removing vertices from a graph

Removing vertices is done in a very similar way to adding them, with the -= operator. Just like the += operator, you can remove a single vertex, or many vertices with the [ ] array notation. These must be identified by their ID:

g -= [(1),(2)];

Once a vertex is removed, all the corresponding edges will also be deleted.

Note: if you try to delete a non existing vertex, nothing will happen.


ID order

All vertices must be added in a sequential order to the graph and the first ID must be 0. This rule is valid for direct declaration of a graph and when adding a vertex. Here are a few example of invalid declarations and their explanation:

g = {1,2,3};

and

g = { };

g += [(1), (2), (3)];

are invalid because they do not start with 0, while:

g = {0,1,3};

and

g = { };

g += [(0), (1), (3)];

are invalid because they do not use the ID 2.
